# Images Directory

This directory is for static images used in the Curling Masters application.

## Usage

Place your image files here and reference them in your templates using:

```html
<img th:src="@{/images/your-image-name.png}" alt="Description">
```

## Recommended Image Types

- PNG for logos and transparent images
- JPG for photographs
- SVG for icons and scalable graphics

## Image Optimization

- Compress images before adding them to this directory
- Use appropriate dimensions for web display
- Consider using WebP format for better compression
